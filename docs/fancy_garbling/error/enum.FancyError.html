pub enum FancyError<T> {
    UnequalModuli,
    NotImplemented,
    InvalidArg {
        desc: String,
    },
    InvalidArgNum {
        got: usize,
        needed: usize,
    },
    InvalidArgMod {
        got: u16,
        needed: u16,
    },
    ArgNotBinary,
    NoTruthTable,
    InvalidTruthTable,
    UninitializedValue,
    ClientError(T),
}

Errors that may occur when using the Fancy trait. ClientError wraps errors from underlying fancy object.

With the exception of ClientError, FancyErrors should be API-usage errors, such as trying to add two Items with different moduli.
