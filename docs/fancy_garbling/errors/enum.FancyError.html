pub enum FancyError {
    UnequalModuli,
    InvalidArg(String),
    InvalidArgNum {
        got: usize,
        needed: usize,
    },
    InvalidArgMod {
        got: u16,
        needed: u16,
    },
    ArgNotBinary,
    NoTruthTable,
    InvalidTruthTable,
    UninitializedValue,
}

Errors that may occur when using the Fancy trait. These errors are API-usage errors, such as trying to add two Items with different moduli.
